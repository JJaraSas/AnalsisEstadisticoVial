Accidentalidad en Bogota

En Bogotá ocurre un siniestro vial cada 5,6 minutos según los datos del de Gestión de Tránsito - CGT y la Secretaría de Movilidad, en cifras tomadas entre Enero y Septiembre de 2019 se presentaron 726 homicidios y 393 muertes por accidentes de tránsito.

Fuente. El Tiempo

¿Que se necesita?

Lo anterior nos permite abrir diferentes incógnitas como por ejemplo que influencia posee un conductor y el tipo de su vehículo en un accidente vehículo, la hora en que se presentan la mayor cantidad de accidentes de tráfico en Bogotá entre otros, por lo cual se propone realizar mediante la ciencia de Datos un análisis estadístico para hallar este tipo de datos de interés, y aplicar el resultado del análisis a la inteligencia artificial y los algoritmos de Machine Learning para predecir accidentes viales en la ciudad con los datos existentes de los factores incidentes.

Solucion Planteada

Inicialmente para la realización del estudio, se deben recopilar datos referentes a siniestros viales en la ciudad de Bogotá, para ello se debe consultar fuentes de datos relevantes como bases de datos gubernamentales, académicas y de interés que brinden información reciente, posteriormente los datos recopilados deben ser organizados, clasificados y depurados.

Teniendo en cuenta lo anterior se debe realizar el tratamiento de datos respectivo para asegurar la calidad de los mismos, este procedimiento se llevará a cabo mediante el lenguaje de programación Python y las librerías que posee en el campo de la ciencia de datos como lo es la librería Pandas. La librería Pandas permite el almacenamiento de datos en las estructuras de datos tabulares conocidas como Data Frames, las cuales están compuestas por columnas y filas ordenadas que facilitan la manipulación de grandes cantidades de datos.

Posteriormente a la realización de la limpieza de los datos, se realizarán cálculos estadísticos para evidenciar información relevante en este tema de estudio, esta información estará acompañada de gráficos estadísticos para facilitar la visualización y entendimiento de los mismos, para ello serán de importancia las librerías Matplotlib que permite generar gráficos a partir de datos obtenidos y Numpy para la creación de vectores en los cuales se almacenen cierto tipo de datos.

El proceso de Visualización de datos brinda una mejor idea sobre aquellos datos con mayor relevancia, los cuales podremos utilizar para la generación de un modelo predictivo a partir de la rama de la inteligencia artificial conocida como machine learning para realizar predicciones de acuerdo a variables específicas, entre ellas una posible predicción será la gravedad del siniestro.

Finalmente se aplica el modelo obtenido con datos sobre accidentes recientes en diferentes bases de datos de la ciudad para asignar una utilidad al modelo realizado, esta etapa es conocida como Despliegue.

  • De acuerdo a lo descrito anteriormente la metodología a llevar a cabo será:
  • Entendimiento de los datos
  • Preparación de los datos
  • Visualización de datos
  • Modelamiento de los datos
  • Evaluación de los datos
  • Despliegue
